# Vibes Player

Vibes Player is a desktop music application built with Electron, Node.js, and TypeScript.  
It is designed to play local music with a modern, minimal, and customizable interface.

## Technologies used

- Electron
- Node.js
- TypeScript
- HTML5 / CSS3
- JavaScript (Renderer process)
- Vite (optional bundler)

## Features (in development)

- Local music playback  
- Play / pause / next / previous controls  
- Playlist system using JSON  
- Modern and customizable UI  
- Volume control  
- Audio visualizer (planned)

## Project goal

This project is a personal practice to improve skills in:

- Desktop application development  
- TypeScript in real-world projects  
- Node.js + frontend integration  
- Modern UI/UX design
